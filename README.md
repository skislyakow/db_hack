# Скрипты для исправления электронного дневника

Набор функций для быстрого исправления оценок, удаления замечаний и добавления похвалы от учителей в базе данных электронного дневника. Предназначен для использования внутри Django-проекта.

## Что умеет этот скрипт:

* **`fix_marks`** — заменяет все плохие оценки (2 и 3) на пятерки.
* **`remove_chastisements`** — удаляет все замечания от учителей из профиля ученика.
* **`create_commendation`** — добавляет случайную похвалу за последний урок по выбранному предмету. Скрипт умный: он не ставит две похвалы на одну дату, чтобы это не выглядело подозрительно.

## Требования

* Django 2.2+ (или совместимая версия).
* Доступ к серверу с развернутым электронным дневником.
* База данных с моделями `Schoolkid`, `Mark`, `Chastisement`, `Commendation`, `Lesson`, `Subject`.

## Установка и запуск

Существует два способа использовать эти скрипты:

### Способ 1: Импорт файла (рекомендуемый)

1. Скачайте файл `hacks.py` и положите его в корневую директорию проекта (рядом с файлом `manage.py`).
2. Запустите консоль Django (Django Shell):
   ```bash
   python manage.py shell
3. Импортируйте функции в консоль:
   ```python
   from hacks import fix_marks, remove_chastisements, create_commendation

### Способ 2: Копирование кода в консоль

1. Открыть файл hacks.py в текстовом редакторе и скопировать всё его содержимое.

2. Запустить консоль Django:
   ```Bash
   python manage.py shell

3. Вставить скопированный код прямо в терминал и нажать Enter. Все функции станут доступны для вызова.

## Примеры использования

### 1. Исправление оценок
Чтобы заменить все двойки и тройки на пятерки для конкретного ученика:
   ```python
   fix_marks('Фролов Иван')
   ```

### 2. Удаление замечаний

Чтобы полностью очистить историю замечаний ученика:
   ```python
   remove_chastisements('Фролов Иван')
   ```

### 3. Добавление похвалы

Чтобы добавить одну похвалу за самый свежий урок (например, по Математике):
   ```python
   create_commendation('Фролов Иван', 'Математика')
   ```

*Если вы запустите эту команду еще раз, скрипт найдет предыдущий урок по этому предмету и добавит похвалу туда, чтобы избежать дублей в один день.*

## Меры предосторожности

Скрипт использует поиск по `contains`, поэтому будьте аккуратны: если вы укажете просто `'Иван'`, и в базе будет несколько Иванов, скрипт выдаст ошибку и попросит уточнить ФИО. Старайтесь указывать фамилию и имя полностью.
